# EJERCICIO SNGULAR

Se ha desarrollado un programa para recorrer un árbol n-ario cuyos nodos son cadenas de texto. El programa devuelve una lista de parejas conteniendo la cadena y el número de ocurrencias de la misma dentro del árbol.

## Requerimientos

- Tener instalado la maquina virtual de Java (JVM).
- Tener instalado un entorno de desarrollo para el lenguaje de programación Java (NetBeans, Eclipse, IntelliJ, etc).

## Intrucciones de compilación y ejecución

- Abrir el proyecto descargado en nuestro entorno de desarrollo.
- Seleccionar opción `Build Project`.
- Seleccionar opción `Run`.

## Localización del código fuente

El código fuente del programa se encuentra en la carpeta: `src/main/`
